Notes on Repositories

  • mandel - Home of the official EOS Network Foundation blockchain node software. Contains the endpoints for the EOSIO RPC API
  • mandel-javascript - Javascript API for integration with EOSIO-based blockchains using EOSIO RPC API
  • mandel-java - EOSIO SDK for Java is an API for integrating with EOSIO-based blockchains using the EOSIO RPC API. This project is compatible with server-side Java and with Android 6+.
  • mandel-swift - EOSIO SDK for Swift is a set of components for integrating with EOSIO-based blockchains.
  • mandel contracts - readme and docs on building contracts updated to v3 NOTE: system contracts 404 (bios, system, msig, token, wrap), lots of broken links
  • mandel appbase - just a readme
  • Mandel Contract Developement Toolkit - CDT is a toolchain for WebAssembly (WASM) and a set of tools to facilitate smart contract development for the Mandel platform. readme and docs
  • Web Assembly Toolkit - readme and man pages
  • Mandel Chainbase - 1 readme. ChainBase is designed to meet the demanding requirments of blockchain applications, but is suitable for use in any application that requires a robust transactional database with the ability have near-infinite levels of undo history
  • Silkworm - 1 readme. ETH VM
